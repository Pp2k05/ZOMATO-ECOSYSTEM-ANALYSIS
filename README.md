# Zomato Ecosystem Sentiment Analysis

## Project Overview
Independent analysis of 45,000+ public data points across Zomato, Blinkit and District — Dec 2025 to Jun 2026.

## Data Sources
Google Play Store | Apple App Store | Reddit

## Data Model
Star schema with 3 tables:
`dim_reviews_core` → `fact_l1_tags`
`dim_reviews_core` → `fact_l2_tags`

## Key Findings
1. Negativity rose from 59% to 67% in 6 months — consistent upward trend
2. Customer Support drives 43.7% of negative reviews not delivery
3. Zomato Gold has 63.28% negative sentiment
4. Premium/Gold Support is #1 Gold complaint — users paid for better support, didn't get it
